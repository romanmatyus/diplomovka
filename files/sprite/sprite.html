Troubleshooting
Sprite Problems

Having trouble getting your Sprite to load in your new website? Here are some tips to sort out what might be the problem.

Sprite is not showing in any browser

This sounds like you need to work on the plumbing. You either did not upload the sprite to the correct directory, or you did not link the sprite properly in the CSS. If you've confirmed that all this is correct and you still have a problem, take a look at your .htaccess file and see if requests are getting intercepted.

WebP is not loading

The most common problem here is that you are serving the sprite from an IIS server. IIS refuses to serve files that have unknown MIME types. If that is the case, you must set the MIME type for WEBP to "image/webp" in the server settings. Follow these instructions from Microsoft if you need help.

Sprite not loading in some browsers

The primary reason for this failure may be that you only have one output format (webp) it's only working in some browser consider editing this sprite and change output format to png or atleast add it as a fallback option
